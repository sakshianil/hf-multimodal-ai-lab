# Project Overview

HF Multimodal AI Lab is a hands-on learning and portfolio project focused on Hugging Face tools for multimodal AI.

## Core areas

1. Transformers foundations
2. AutoTokenizer and AutoModel usage
3. High-level `pipeline()` API
4. Image captioning and visual question answering
5. Speech-to-text workflows
6. Diffusers and image generation
7. Hugging Face inference endpoint usage
8. Gradio / Spaces deployment planning

## Main showcase workflow

The first highlighted workflow is:

```text
Hindi singing audio
  ↓
Speech-to-text transcription
  ↓
Prompt/image-generation step
  ↓
Generated image output
```

This demonstrates multimodal chaining: audio is transformed into text, and text is then used as the signal for a generative visual model.

## Portfolio angle

This project is useful for showing applied AI engineering skills because it combines notebooks, model APIs, multimodal inputs, generative outputs, and deployment planning.
